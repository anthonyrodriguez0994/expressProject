Project File Structure

./app.js
    Primary express app file.

./*-helper.js
    Server-side JS function definitions.  Unit-tested.

./routes/*.js
    Server-side http request-handling code.  Core functionality.

./views/*.pug
    Pug dynamic-html templates.  Rendered by a call to res.render() as the final step of routing code

./test/*-helper.js
    Mocha unit tests for the ./*-helper.js file of the same name, plus per-module setup and teardown code.

./test/zzz_teardown.js
    Unit-test teardown code - closes the db connection opened when unit tests are run

./public/javascripts/*
    Client-side JS code

./public/stylesheets/*
    CSS for pages generated by pug
